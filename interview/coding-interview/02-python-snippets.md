## 如果有一个数为0，返回另一个

```py
if n * m == 0:
    return n + m
```

##  生成数组

```py
dp = [False for _ in range n]
dp = [False] * n
```


## 生成 m*n 的空矩阵

常用在二维 DP 中

```py
d = [ [0] * m for _ in range(n)]
```
