## 分治的时间复杂度


![](https://tva1.sinaimg.cn/large/00831rSTly1gd19dz03f1j308c062ac6.jpg)

通过O(n)的时间,把n的问题,变为了两个n/2的问题,复杂度是多少?

    T(n) = 2T(n/2) + O(n)
        = 2 * 2T(n/4) + O(n) + O(n)
        = n + nlogn

通过O(1)的时间,把n的问题,变成了两个n/2的问题,复杂度是多少?

    T(n) = 2T(n/2) + O(1)
    = 2 * 2T(n/4) + O(1) + O(1)
    = n + (1 + 2 + 4 +…+ n)
    ≈ n + 2n
    ≈ O(n)
