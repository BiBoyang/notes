# 可变与不可变

Rust 中使用 let 来绑定一个变量，默认是不可变的。也就是说默认就相当于 C++ 中 const 变量。但是实际上 C++ 中的 const 也只是只读而已。

```
let x = 5;
x = 6;  // 非法
```
如果想要更改一个变量的值，在声明的时候，应该加上 mut（每次写 let mut 的时候，我心里都是读作“让TM的”）

```
let mut = 5;
x = 6
```
