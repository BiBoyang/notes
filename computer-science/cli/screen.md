tmux 比 screen 好用多了，放弃 screen 了

screen is old and outdated, but for the sake of making money, I have to use it...

Creating a session

`screen -S name`

reentering a session

`screen -r name`

detach

`prefix d`

create a new window

`prefix c`

kill current window

`prefix k`

goto last window

`prefix C-a`

change window

`prefix number`

copy mode

`prefix [`
